Términos y Condiciones de Servicio

Última actualización: Junio 2025

Bienvenido a Info.2B. Al utilizar nuestros servicios, aceptas los siguientes términos y condiciones. Es importante que los leas detenidamente antes de enviar una solicitud de servicio o utilizar nuestras herramientas.

1. Naturaleza del servicio

Info.2B es una plataforma orientada al soporte técnico, diagnóstico y reparación de equipos electrónicos (PC, laptops, redes, etc.). El usuario puede crear solicitudes en línea y recibir un código de seguimiento único para consultar el estado de su equipo.

2. Veracidad de los datos

El usuario se compromete a proporcionar datos verídicos, actualizados y completos en el formulario de contacto. Info.2B no se responsabiliza por retrasos o errores derivados de información incorrecta.

3. Responsabilidad de la información

Al enviar una solicitud, el usuario autoriza a Info.2B a recopilar y almacenar los datos necesarios para brindar el servicio. Esta información se utiliza exclusivamente con fines técnicos y administrativos.

4. Diagnóstico y reparación

  • Los diagnósticos básicos son gratuitos, salvo que se indique lo contrario.
  • Todo trabajo de reparación se realizará previa autorización del cliente.
  • Info.2B no se hace responsable por daños ocultos o defectos preexistentes.

5. Garantías

Las reparaciones pueden tener garantía limitada según el componente intervenido. Las condiciones de garantía se detallan al momento de entregar el equipo.

6. Código de seguimiento

El usuario recibirá un código único al enviar su solicitud. Es responsabilidad del cliente conservar y usar ese código de forma segura. El acceso no autorizado a este código puede comprometer la confidencialidad del caso.

7. Canales de contacto

El usuario podrá elegir cómo recibir su código (correo o WhatsApp). Posteriormente podrá recuperar dicho código mediante una página segura de validación.

8. Uso del sitio
